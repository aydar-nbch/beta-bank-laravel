# Веб-приложение «Бета-Банк»

Сервис для расчета финансовых продуктов (кредитов и вкладов) с панелью администратора. Разработано в рамках учебной практики (4 семестр).

# Основные возможности:
* **Кредитный калькулятор**: расчет ежемесячного платежа, необходимого дохода и общей переплаты.
* **Депозитный калькулятор**: расчет доходности вкладов.
* **Админ-панель**: просмотр истории расчетов, управление логами и анализ статистики.
* **Безопасность**: защита от CSRF-атак, SQL-инъекций и массового заполнения данных.
* **SEO-оптимизация**: семантическая разметка, мета-теги и управление индексацией (robots.txt).

# 🛠 Технологический стек:
* **Framework**: Laravel
* **Language**: PHP
* **Database**: MySQL
* **Frontend**: Blade, HTML, CSS, Vite

# Инструкция по развертыванию:
1. Клонировать репозиторий:  
   `git clone https://github.com/aydar-nbch/beta-bank-laravel.git`
2. Установить зависимости:  
   `composer install`  
   `npm install && npm run build`
3. Подготовить файл окружения:  
   `cp .env.example .env`  
   `php artisan key:generate`
4. Настроить базу данных в `.env` и запустить миграции:  
   `php artisan migrate --seed`
5. Запустить локальный сервер:  
   `php artisan serve`
